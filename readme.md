# Finite automaton

    This project simulates a **Finite automaton** which was developed in Python 3.14.2 for the subject **Formal Languages**

    The automaton

    ## Requirements
    - Python 3.14.2
    - Tested on Linux (Manjaro)

## Usage

    Run the script:
    python automaton.py 

### Usage Examples
    Example (Accepted)
    Insert the string for the automaton: a
    This is the shorter example for a accepted status

    Example (Accepted)
    Insert the string for the automaton: aaaabb
    
    Example (rejected)
    Insert the string for the automaton: aaaa
    This is the rejected status in the automaton.
